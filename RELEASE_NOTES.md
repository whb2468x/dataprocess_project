# DataProcess Flex v1.0.0

## 🎉 Initial Release

DataProcess Flex is a modular, extensible data processing toolkit for tabular and grouped data. This is the first stable release with full functionality.

### ✨ Features

- **Data Processing Operations**: Clean, normalize, filter, and transform tabular data
- **Interactive Mode**: User-friendly command-line interface for data exploration
- **Pipeline Support**: JSON-based batch processing for repeatable workflows
- **Statistical Analysis**: Aggregation, comparison, and statistical operations
- **Multi-format Support**: Handle CSV, TSV, and other delimited formats
- **Extensible Architecture**: Modular design for easy extension

### 🛠️ Technical Details

- **Python Version**: >= 3.10
- **Dependencies**: numpy >= 1.26, pandas >= 2.2
- **License**: MIT
- **CI/CD**: GitHub Actions with Python 3.10 and 3.11 testing

### 📦 Installation

```bash
pip install git+https://github.com/whb2468x/dataprocess_project.git@v1.0.0
```

Or from source:
```bash
git clone https://github.com/whb2468x/dataprocess_project.git
cd dataprocess_project
pip install -e .
```
