# Contributing to DataProcess Flex

Thank you for your interest in contributing to this project.

## How to contribute

1. Fork the repository.
2. Create a branch for your feature or fix: `git checkout -b feature-name`.
3. Make your changes and include tests where applicable.
4. Run the unit tests locally:

```bash
python -m unittest discover -s tests
```

5. Commit your changes and open a pull request.

## Reporting issues

Please open an issue for bugs, feature requests, or documentation improvements.

## Coding style

Keep the code readable and consistent with the existing Python style in the repository.
